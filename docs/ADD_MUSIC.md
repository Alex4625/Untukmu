# Menambahkan Musik

Browser biasanya memblokir autoplay, jadi website ini memakai tombol play manual.

## Cara termudah

1. Masukkan file musik ke folder:

```text
public/audio/
```

Contoh:

```text
public/audio/lagu-kecil.mp3
```

2. Jalankan website.
3. Buka `/admin`.
4. Masuk tab **Settings**.
5. Isi URL Musik:

```text
/audio/lagu-kecil.mp3
```

6. Klik **Simpan Settings**.

## Catatan

Gunakan file kecil agar website tetap ringan. Jangan upload musik yang melanggar hak cipta kalau website akan dibagikan publik.
